# Travel Splitter | 旅行分帳系統💰

Full-stack expense-splitting app for group trips with real-time tracking and automated settlement calculation.

---

## 🇺🇸 English

### Overview
Travel Splitter is a web application designed to simplify expense sharing during group trips. It allows users to record expenses, track balances in real time, and automatically compute optimal settlements to minimize the number of transactions required.

### Tech Stack
Next.js · React · Node.js · Express · MongoDB

### Features
- Real-time group expense tracking and balance updates  
- Automated debt optimization to compute minimal who-owes-whom transactions  
- Multi-user trip system with shared financial records  
- Responsive UI for creating trips, adding expenses, and viewing settlements  
- RESTful API integration with persistent database storage  
- Deployed using GitHub with Render/Vercel

## 🇹🇼 中文

### 概述
Travel Splitter 是一個旅行分帳網站應用，協助使用者在多人旅行中記錄支出、即時查看帳務狀態，並自動計算最優結算方式，減少轉帳次數。

### 技術棧
Next.js · React · Node.js · Express · MongoDB

### 功能
- 即時多人分帳與帳務更新  
- 自動債務最佳化，計算最少轉帳次數  
- 多人行程系統，支援共享財務紀錄  
- 響應式介面（建立行程、輸入支出、查看結算）  
- REST API 與資料庫持久化整合  
- 使用 GitHub 版本控制並部署至 Render / Vercel
